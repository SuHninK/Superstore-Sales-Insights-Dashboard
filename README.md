# Superstore-Sales-Insights-Dashboard
A Professional Power BI Dashboard focused on sales trends, regional distribution, and delivery logistics optimization.

## Project Overview
This interactive Power BI dashboard provides deep strategic insights into business performance, regional growth, and product segment distribution. Built around dynamic user interactions, it helps operational managers optimize sales strategies and streamline deliveries.


### Key Metrics Tracked (As of Dashboard):
- **Total Sales:** $2.30M (Represented as 2M in KPI card)
- **Total Orders:** 4,922 Orders
- **Average Delay Days:** 4 Days
- **Average Order Value:** $459

---

## Key Business Insights (Strategy Focus)
- **Regional Breakdown (Small Multiples):** Using optimized small multiples, sales trend over time performance is monitored seamlessly across Central, East, South, and West regions.
- **Segment Analysis:** Clear visual breakdown shows high performance within the Consumer, Corporate, and Home Office segments across top product categories (Technology, Furniture, and Office Supplies).
- **Geographical & Logistics Distribution:** The integrated map visualizes geographical sales density, perfectly pairing with the Ship Mode breakdown to highlight operational efficiencies.

---

## Technical Implementation Details
- **Data Engineering:** Extracted, cleaned, and transformed raw transactional datasets using Power Query.
- **Data Modeling:** Established a streamlined star-schema model for fast data processing.
